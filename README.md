#  K-Term

Un emulador de terminal experimental acelerado por GPU, diseñado con una arquitectura de bloques estructurados y navegación modal.

>  **WIP (Work in Progress) / Pre-Alpha:** Este proyecto está en fase de exploración inicial. El código es un borrador y la arquitectura cambiará constantemente. 

##  Características Planeadas

*   **Renderizado por GPU:** Construido sobre `wgpu` para un rendimiento sin cuellos de botella.
*   **Arquitectura de Bloques:** Aislamiento de la salida estándar (stdout/stderr) por cada comando ejecutado, facilitando la copia y re-ejecución.
*   **Navegación Modal:** Flujo de trabajo pensado para el teclado, permitiendo moverse entre bloques sin tocar el ratón.
*   **Estética Nivelada:** Soporte nativo para shaders y un diseño por defecto inspirado en interfaces industriales y *cyberpunk noir* (alto contraste, acentos en rojo neón).
